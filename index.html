How to play squares?

The aim of the game is to arrange the numbers on the board in ascending order from left to right and top to bottom, as you see them before you shuffle the board by starting the game.

You can do this by either clockwise or counterclockwise rotating the numbers highlighted in red.

You can move the selection (rotator) around using the arrow or WASD keys. You can rotate it clockwise using full-stop or 'E' and counterclockwise using comma or 'Q' keys.

If you're using a touch device, you can drag the selection using a swipe gesture from a red number to a black one and rotate it either way by swiping from a red number to another red one.

Before starting a game, you get set the parameters (hence somewhat the difficulty) of the puzzle in the menu on the left. Each unique combination of parameters should give you the same puzzle each time you play that set, so you can compete with yourself and others.
